Qu'est-ce qu'une catégorie ?

Les catégories permettent de classifier les différents types de travaux réalisés dans vos ateliers (soudure, usinage, assemblage, etc.).

Conseils :
  • Choisissez un nom clair et explicite
  • Évitez les doublons
  • Décrivez précisément les types d'activités
  • Une bonne catégorisation facilite la planification
Astuce : Les catégories seront utilisées pour associer les collaborateurs aux compétences et les ateliers aux capacités.
Exemples de catégories
